### Project Description
A simple interface that calculates travel expenses using starting point,destination,vehicle mileage and fuel cost.It estimates distances and shows total petrol and travel cost instantly.

### The Problem statement
Poeple find it hard to know travel costs before a trip We need a simple tool to quickly calculate total cost using distance ,mileage and fuel price.

### The Solution
we solve it by creating a simple interface where users enter trip details like distance,mileage,and fuel price.It instantly calculates and shows the total travel cost.
